# TinyLink - URL Shortener

A full-stack URL shortener built with Node.js, Express, PostgreSQL, and Tailwind CSS.

## Features

- ✅ Shorten URLs with custom codes
- ✅ Redirect with click tracking
- ✅ Dashboard with link management
- ✅ Statistics for individual links
- ✅ Responsive design
- ✅ Health check endpoint
